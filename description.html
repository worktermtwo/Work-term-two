Job Description

When applying for this position, the job was described as follows:

Applications programmers write code for software applications. Tasks may include:

Application Development
· Researching and examining current systems
· Conversion of existing applications to .Net framework
· Web authoring using WordPress
· Knowledge of Windows Servers 2008+
· Writing software and operating manuals
· Product testing
· Making appropriate modifications to existing applications as need be.

Computing Support
· Support in managing VM servers
· Support in setting up workstations for end users
· Software installation support
However this is a much more abstract approach to my job duties. Realistically, I would have to assist in taclking whatever projects the department undertook over
the course of the summer. I spent most of my time learning new tools and technologies (.Net, PHP, HTML/CSS, C# etc..) so that I could find ways of contributing.
If asked to summarize this job position, I would say that it is working independently to find creative soloutions to dynamic problems, using .Net technologies.
